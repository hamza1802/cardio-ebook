# Cardio: The Power of Movement

This is a static website for the "Cardio: The Power of Movement" ebook. It includes detailed workout plans for Beginner, Intermediate, and Advanced levels.

## Structure

- `ebook.html`: The main landing page.
- `beginner.html`: Detailed beginner plan.
- `intermediate.html`: Detailed intermediate plan.
- `advanced.html`: Detailed advanced plan.
- `styles.css`: Global styles.

## Deployment

This project is ready to be hosted on any static site hosting service, such as **Azure Static Web Apps**, **GitHub Pages**, or **Vercel**.
